プライバシーポリシー

タコスコ（以下「当サイト」）は、ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

当サイトでは、以下の情報を取得する場合があります。

・ご利用のブラウザから自動的に送信される情報（IPアドレス、ブラウザの種類、アクセス日時等）

・Cookie を用いた情報（アクセス解析・広告配信のため）

・ユーザーが任意で入力した口コミ情報

2. 情報の利用目的

取得した情報は以下の目的で利用します。

・サイトの運営・改善

・アクセス解析による利便性の向上

・広告の配信・最適化

3. 広告について

当サイトでは、第三者配信の広告サービス（Google AdSense）を利用しています。広告配信事業者は、ユーザーの興味に応じた広告を表示するために Cookie を使用することがあります。

Google AdSense に関する詳細は、Google 広告ポリシーをご覧ください。

ユーザーは、Google の広告設定ページ（https://www.google.com/settings/ads）で、パーソナライズ広告を無効にすることができます。

4. アクセス解析ツール

当サイトでは、Googleによるアクセス解析ツールを使用する場合があります。このツールはトラフィックデータの収集のためにCookieを使用しています。データは匿名で収集されており、個人を特定するものではありません。

5. データの保存

当サイトで追加された店舗情報・口コミデータは、ユーザーのブラウザのローカルストレージに保存されます。サーバーへの送信は行いません。

6. ポリシーの変更

本ポリシーの内容は、必要に応じて変更することがあります。変更後のプライバシーポリシーは、当サイトに掲載した時点より効力を生じるものとします。

最終更新日: 2026年3月21日

利用規約

本利用規約（以下「本規約」）は、タコスコ（以下「当サイト」）の利用条件を定めるものです。

1. サービス内容

当サイトは、全国のタコス専門店の情報を掲載し、ユーザーが口コミ・評価を共有できるグルメ情報サイトです。

2. 掲載情報について

当サイトに掲載される店舗情報は、Google Places API および手動調査により収集しています。情報の正確性には十分注意を払っていますが、営業時間・メニュー・価格等は変更される場合があります。最新情報は各店舗に直接ご確認ください。

3. 口コミ投稿

ユーザーが投稿した口コミは、ローカルストレージに保存されます。誹謗中傷・虚偽の内容・公序良俗に反する内容の投稿はご遠慮ください。

4. 免責事項

当サイトの利用により生じたいかなる損害についても、当サイト運営者は一切の責任を負いません。掲載情報を参考にされる際は、ご自身の判断と責任においてご利用ください。

5. 著作権

当サイトに掲載されるコンテンツ（テキスト、画像、デザイン等）の著作権は当サイト運営者に帰属します。無断転載・複製を禁じます。

6. 規約の変更

本規約は、必要に応じて変更することがあります。変更後の規約は、当サイトに掲載した時点より効力を生じるものとします。
